Contributing
============

LocalWiki is an open-source project, and we need your contributions to make
it even more awesome.  If you see any bugs, please let us know.  If you want
to contribute a patch, even better!

Everything you need to contribute can be found at https://dev.localwiki.org

The fastest way to reach the main developers is on the `#localwiki IRC channel
<irc://irc.freenode.net/#localwiki>`_
